# Nikki — Nike Sneaker Consultant

AI-powered sneaker stylist built with React + Claude API.

## Setup

1. Install dependencies: `npm install`
2. Add your Anthropic API key in Vercel environment variables as `ANTHROPIC_API_KEY`
3. Deploy to Vercel

## Local development

Create a `.env.local` file:
```
ANTHROPIC_API_KEY=sk-ant-...
```

Then run: `npm run dev`
